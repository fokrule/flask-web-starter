About This Project

This is a basic Flask web application designed to demonstrate fundamental web development concepts in Python.

It includes:

  • Modular application structure using Blueprints.
  • Basic routing and view rendering.
  • Simple user authentication (login/logout).
  • HTML templating with Jinja2.
  • Dynamic content generation (Random Quote Generator).
